# Usage

Create an island

> deno run -A https://deno.land/x/juicer/juicer.ts island MyButton

Create a route

> deno run -A https://deno.land/x/juicer/juicer.ts route posts

# Install

> deno install -A https://deno.land/x/juicer/juicer.ts
